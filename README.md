# ahkD-dvekt
Ahk skript for å raskt fylle ut dødvektstrykk i excel.
<br />
<br />
Ctrl + Numpad 0 = Enable <br />
Ctrl + Numpad , = Bytt mellom 1/8 inch og 1/80 inch <br />

<br /> 
Ctrl + Numpad [n] for å fylle inn vekt til følgende lodd:<br />
Numpad 1 = L<br />
Numpad 2 = 0,5<br />
Numpad 3 = 1<br />
Numpad 4 = 2 A<br />
Numpad 5 = 2 B<br />
Numpad 6 = 5<br />
Numpad 7 = 9<br />
Numpad 8 = 10 A<br />
Numpad 9 = 10 B<br />
Numpad / = 10 C<br />
Numpad * = 10 D<br />
 <br />
