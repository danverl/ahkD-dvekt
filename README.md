# ahkD-dvekt
Ahk skript for å raskt fylle ut dødvektstrykk i excel.
/n
Ctrl + Numpad 0 = Enable /n
Ctrl + Numpad , = Bytt mellom 1/8 inch og 1/80 inch /n

/n 
Ctrl + Numpad [n] for å fylle inn vekt til følgende lodd:
Numpad 1 = L
Numpad 2 = 0,5
Numpad 3 = 1
Numpad 4 = 2 A
Numpad 5 = 2 B
Numpad 6 = 5
Numpad 7 = 9
Numpad 8 = 10 A
Numpad 9 = 10 B
Numpad / = 10 C
Numpad * = 10 D
